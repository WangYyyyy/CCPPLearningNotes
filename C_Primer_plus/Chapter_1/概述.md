### C语言基本概念

1. 运算符: =

2. 函数: main(), printf()

3. \#include

4. 转义序列 \n \t \b等

5. 自定义函数

   1. 声明函数体 

      main函数前

      ```
      void myFunction(void);
      ```

   2. 函数定义

      main函数后

      ```
      void myFunction(void) {
          //函数体
      }
      ```

   3. 函数调用

      main函数中

      ```
      int main(void) {
          //codes ...
          myFuntion();
          //codes...
          return 0;
      }
      ```
