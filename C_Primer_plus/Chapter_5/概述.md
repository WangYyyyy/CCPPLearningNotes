### 实参和形参

 * 实参

   actual argument or actual parameter 

   是调用函数传入的参数

   function(*actual argument)

* 形参

  formal argument or formal parameter 

  是函数中的赋给相应的值

  function(argument) {

  ​	*formal argument

  }
